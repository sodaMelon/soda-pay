## soda-pay
간편 결제 웹서비스 "소다 페이" (Simple purchase system on Web)

금융결제원 오픈뱅킹 테스트베드 API를 이용하여, 실제 휴대폰 본인 인증을 거친 [간편 결제 서비스]를 이용할 수 있습니다.
(유사 서비스: 삼성 페이, 네이버 페이, 페이코 등....)
